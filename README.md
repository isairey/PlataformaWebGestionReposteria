# 🌌 Acerca del proyecto

**Web Cake Shop** es una plataforma web enfocada en la venta online de tortas, postres y productos de repostería.

El sistema permite:

- 🍰 Mostrar catálogo de productos
- 🛒 Gestionar carrito de compras
- 📦 Visualizar productos destacados
- 📱 Navegación responsive
- ⚡ Gestión de estado moderna
- 🎨 Interfaz atractiva y dinámica
- 🧁 Organización de categorías
- 🚀 Experiencia SPA con Vue 3

# ⚡ Instalación

## 📋 Requisitos

- Node.js v14 o superior
- NPM
- VS Code
- Git

---

# 🚀 Configuración del proyecto

## 1️⃣ Clonar repositorio

```bash
git clone https://github.com/usuario/web-cake-shop.git
```

---

## 2️⃣ Entrar al proyecto

```bash
cd web-cake-shop
```

---

## 3️⃣ Instalar dependencias

```bash
npm install
```

---

## 4️⃣ Ejecutar servidor de desarrollo

```bash
npm run dev
```

Abrir navegador en:

```bash
http://localhost:5173
```

---

## 5️⃣ Ejecutar linter

```bash
npm run lint
```

---

## 6️⃣ Compilar proyecto

```bash
npm run build
```
